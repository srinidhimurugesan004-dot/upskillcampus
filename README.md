## Smart IoT-Based Theft Detection and Alert System

## Overview
This project is an IoT-based theft detection system developed using ESP32 and PIR motion sensor.

The system detects motion and sends real-time data to cloud platforms using MQTT protocol.

## Features
- Motion detection using PIR sensor
- Buzzer alert system
- ESP32 WiFi connectivity
- MQTT cloud communication
- Real-time monitoring

## Technologies Used
- ESP32
- Embedded C
- Arduino IDE
- MQTT
- ThingsBoard
